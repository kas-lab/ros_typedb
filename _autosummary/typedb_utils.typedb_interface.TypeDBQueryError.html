typedb_utils.typedb_interface.TypeDBQueryError

exception typedb_utils.typedb_interface.TypeDBQueryError(*, session_type, transaction_type, query_type, query)

Raised when a TypeDB query fails during execution.

Build a contextual error for failed TypeDB query execution.

Parameters:

  • session_type (str) – Requested API session type (schema/data).

  • transaction_type (str) – Requested transaction mode (read/write).

  • query_type (str) – Logical query kind (insert/get/fetch/etc.).

  • query (str) – Original TypeQL query text.

Returns:

None.
